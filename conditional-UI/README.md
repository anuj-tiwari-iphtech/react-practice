# Conditional UI with State

This project demonstrates how to use React's `useState` hook to conditionally render UI based on state changes.

## Topics Covered

- Loading State
- Login / Logout
- Toggle Components

## Practice Exercises

- Login Button
- Dark / Light Mode
- Show More / Show Less

## Learning Objectives

By completing these examples, you will learn how to:

- Use `useState` to manage UI state
- Render different UI based on conditions
- Use the ternary operator (`condition ? A : B`)
- Use logical AND (`condition && element`) for conditional rendering
- Build interactive and dynamic React components

## Technologies Used

- React
- JavaScript (ES6+)
- JSX

## Getting Started

```bash
git clone https://github.com/anuj-tiwari-iphtech/react-practice.git
```

### Navigate to the project folder

```bash
cd conditional-UI
```


1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm run dev
```

or, if using Create React App:

```bash
npm start
```
